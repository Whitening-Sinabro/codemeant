## Claude 필수 행동 지침

### 1. 파일 변경 시 업데이트
- 새 파일 생성/삭제/이동 시 → "주요 파일" 섹션 즉시 업데이트

### 2. 플랜 제시 시 크리틱 제공
사용자가 플랜을 제안하면 반드시 크리틱과 해결안 제공:

```
## [플랜명] 크리틱

| # | 문제점 | 심각도 | 해결안 |
|---|--------|--------|--------|
| 1 | [문제] | 높음/중간/낮음 | [해결안] |

**권장**: [최종 권장 사항]
```

### 3. 큰 파일 읽기 규칙
파일이 너무 커서 한 번에 읽을 수 없는 경우:
- **금지**: 중요 부분만 읽고 나머지 스킵
- **필수**: 읽을 수 있는 분량씩 나눠서 **전체 파일** 읽기

**이유**: 전체 파일을 알아야 수정이 용이함. 일부만 알면 반복 수정 발생.

**행동 패턴**:
```
1. offset=0, limit=2000 으로 첫 부분 읽기
2. offset=2000, limit=2000 으로 다음 부분 읽기
3. ... 파일 끝까지 반복
4. 전체 구조 파악 후 수정 시작
```
